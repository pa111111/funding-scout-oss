База на проде копит с 2026-05-05 04:02 UTC. К 2026-05-08 (3 дня) уже было видно несколько неочевидных паттернов.

## Главный сюрприз: cross-DEX same-ticker — это **windows**, не carry

Гипотеза которую делали в продуктовой части: "спред funding APR стабилен часами-сутками, заходишь и держишь carry".

**Реальность по BTC HL ↔ Lighter за 20 часов 2026-05-07/08:**

```
05-07 14:05    HL +6.8%    Lighter +56%   spread -49%   ← окно отрицательного spread
05-07 15:05    HL +9.0%    Lighter +70%   spread -61%
05-07 16:05    HL +11%     Lighter +70%   spread -59%
05-07 17:05    HL +11%     Lighter +84%   spread -73%
05-07 18:05    HL +11%     Lighter +84%   spread -73%   ← 5 часов держался -60–73%
05-07 19:05    HL +11%     Lighter   0%   spread +11%   ← ПЕРЕВОРОТ
05-07 20:05    HL +11%     Lighter +14%   spread -3%
05-07 21:05    HL +4.3%    Lighter -133%  spread +138%  ← БОЛЬШОЙ рывок в обратку
05-07 22:05    HL +4.5%    Lighter -105%  spread +110%
```

Паттерн: **spread сидит в одном направлении 4-12 часов, потом резко переворачивается**. Если зайти и забыть на сутки — выйдешь в минусе из-за ротации.

**Что это значит для продукта:**

1. **Стратегия "ставлю 80% APR carry и держу неделю" — НЕ работает** на большинстве cross-DEX same-ticker связок.
2. **Реальная стратегия — поймать окно и закрыться когда spread сжимается до нуля.** Удержание 4-12h в позитив, выход.
3. **Это меняет приоритет фич:**
   - 🔴 **Stability column** становится критичной — % времени за 24h когда spread держится одного знака. Без неё пользователь не отличит "window" от "carry".
   - 🔴 **History chart drill-down** — увидеть глазами что это окно, а не вспышка минутная.

См. также: `next_features.md` если он существует, иначе `mvp_v01_state.md` для roadmap.

## Lighter и Pacifica применяют clamp/floor APR на equity

В первом 4-venue snapshot'е (после деплоя Pacifica 2026-05-08):

```
Lighter equity APR (все тикеры):  +28% APR  ← застывшая floor
Pacifica equity APR (все тикеры): +13.1% APR ← застывшая floor

Тикеры с этим паттерном: NVDA, TSLA, PLTR, HOOD, GOOGL, AAPL, MSFT (если есть)
```

Интерпретация: обе биржи применяют **clamp** на funding для equity-перпов — видимо потому что equity не имеет «свободного рынка маркет-мейкеров» в выходные/ночью как крипта, и rate приходится фиксировать административно.

**Что это значит:**

1. **Equity cross-DEX spread ≈ постоянный 15% APR** между Lighter (+28%) и Pacifica (+13.1%) — **реально carry**, не window. Это первый класс связок где carry работает.
2. **Но 15% APR — скромно**. На $5k = $750/год = $63/мес. Без эффекта плеча.
3. **EdgeX тоже листит equity** и теоретически дал бы третью точку — но он часто заблокирован 403'кой на equity по IP (регуляторное).
4. **Pacifica AMZN +343% APR** наблюдался без второй ноги — связки не получилось. Если когда-то ещё одна биржа подключит AMZN — это будет очень интересный setup для backtest.

## Crypto BTC/ETH/SOL — что реально торгуется на triangulation

```
BTC   HL +11%  / Lighter   0%  / Pacifica +2.4%   → HL-Lighter spread +11% (узкий)
ETH   HL +11%  / Lighter -28%  / Pacifica +5.3%   → HL-Lighter spread +39% (средний)
SOL   HL  -8%  / Lighter +84%  / Pacifica +1.5%   → Lighter-HL spread +92% 🔥
```

SOL с двусторонней ликвидностью ($330M HL + $54M Lighter) — реальная торгуемая связка прямо сейчас. **Это реальный target stratегии**, а не теория. Но см. выше — это спред в моменте, может перевернуться за 4-12 часов.

## Pacifica funding ровный (~+2% APR base) на всём crypto

Pacifica на BTC/ETH/SOL имеет **низкие и плавные** funding rates (BTC +2.4%, ETH +5.3%, SOL +1.5%). Это значит:

- Pacifica useful как **третья точка triangulation**, но spread c HL/Lighter обычно мелкий
- На equity у них тот же clamp +13.1% что и crypto +2.4%? Нет, разные числа — но обе плавные
- Pacifica скорее всего сравнительно тонкая в OI (BTC OI = $35M vs HL $400M+), потому что funding не разъезжается активно

## EdgeX intermittent — но не настолько плох

После initial rate-bann (когда мы интенсивно пробинговали API с лаптопа) — EdgeX **работает в большинстве** automated snapshot'ов на VPS. Из 73 primary контрактов:

- **Crypto** (~52-54 ticks): доступны почти всегда
- **Equity** (~19-21 контрактов): 403'аются ВСЕГДА из Германии (региональное ограничение — это не bug, это regulatory)

**Workaround на будущее**: если жгуче нужны equity на EdgeX, нужен другой VPS в юрисдикции где EdgeX отдаёт equity (вероятно вне ЕС). Пока — Pacifica закрывает equity-сегмент.
