## Четыре источника edge

| Источник | Что это | Как помогает продукт |
|---|---|---|
| **Time alpha** | Связки на новых DEX живут 2-4 недели до прихода маркетмейкеров. Кто увидел первым — забрал. | Сканит все интегрированные venues разом, ты не ходишь по 5 UI |
| **Capacity alpha** | $5k капитал влезает в связки с ёмкостью $30k, куда фонды на $5M не могут | Колонка `Min vol 24h` показывает где играют киты, а где можешь только ты |
| **Information alpha** | Никита/Дарис в Telegram палят сетапы. Большинство ленится перепроверять — ты ставишь свой filter | Ты видишь те же связки и сравниваешь с каналом за минуту |
| **Discipline alpha** | НЕ ловишь 1000% APR на пустом стакане. Берёшь 80% на BTC и держишь по правилам | Парадигма "показываем всё, не фильтруем" — учишься различать связку от капкана |

Продукт даёт 1, 2, 3. **Discipline — на пользователе.**

## Ожидаемая доходность (без хайпа)

Из практики Никиты в стриме:
- **50-100% APR baseline** на дистанции
- **+20-30% за 2 недели** на свежей DEX (time-alpha окно)
- **−6-7% за день** в flash-crash (10.10.2025 эталон)

Прикидка для $5k капитала:
- Норма: ~$70/нед = ~$3500/год = **+70% годовых**
- Сильный месяц: +$400
- Плохой месяц: −$300 (один flash-crash)
- Годовая просадка макс: ~10-15% если соблюдаешь правила
- Сравнение: SP500 ~10%, treasury ~4%

```
Вечер (5 мин):
  - Позиции живы? funding капал?
  - Связка ушла против → закрыл и забыл. НЕ ждём "вернётся".
```

## Системные НЕТ

- ❌ Не ловишь honeypot (ZRO 600% APR на $0.5M)
- ❌ Не средствуешь убытки ("докину чтобы выровняться")
- ❌ Не открываешь по интуиции — только если прошло правила
- ❌ Не FOMO на алерт Никиты — сначала свой filter
- ❌ Не пытаешься перебежать ногу с биржи на биржу

## Slippage — главная формула (вместо моего ошибочного $10M heuristic)

Я в один момент сказал "Min vol $10M = реальная связка, меньше — honeypot". Это было **неверно** — порог был для $5k, не универсальный. Правильная формула:

```
slippage % ≈ order_size / hourly_volume × k
                                          ↑
                                   k ~ 0.5–2 (форма стакана)
```

Прогон:
- $5000 на ZRO ($23k/h vol) → 21% slippage 🔴 honeypot
- $1000 на ZRO → 4% — на грани
- $300 на ZRO → 1.3% 🟢 пройдёт за день фандинга
- $5000 на BTC ($44M/h) → 0.011% невидимо

**Это рассказано пользователю, продукт пока эту формулу не считает явно** (это будущая фича: capital input + slippage column).

## Минимальное правило начала

> **Открой ровно 1 связку с min vol > $50M (BTC/ETH/SOL), $1000 капитала, плечо 2x. Записал вход. Через 7 дней посмотрел PnL. Зачти результат — даже −2% это победа, потому что прошёл цикл и не запаниковал.**

После 4-5 таких циклов появится feel. Потом усложняешь (больше связок, плечо, мельче ликвидность). Не наоборот.

## Главный takeaway

**Продукт не делает богатым.** Он:
1. Сокращает время на поиск с 30 минут до 1 минуты
2. Подсвечивает контекст (vol, slippage, риск) чтобы не наступать на грабли

Богатым делает **стабильное применение правил на дистанции 6-12 месяцев**.
